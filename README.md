# Pearl – Personal Kit Plugin for CPvP Servers

**Pearl** is a lightweight and powerful PvP personal kit plugin that lets players manage and use their own custom kits easily.

---

## ⚔️ Features
- 🧰 Personal kit system for each player  
- ⚡ Fast and optimized performance  
- 🎯 Perfect for PvP servers  
- 🔄 Easy to use commands  
- 🧩 Plug-and-play setup  

---

## 🛠️ Requirements
- Java 17+  
- Maven  
- Server software:
  - Paper / Spigot / Folia

---

## How to edit kit room (For Admins)
- Right click on the Kit room
- Keep items and click on save

---

## How to edit premade kits (For Admins)
- Right click on the kit you want to edit
- Click on import (imports inventory)
- Save

---
